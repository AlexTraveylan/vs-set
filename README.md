# Vs-set : vs-code settings generator

A rust tool for create .vscode folder with settings.json very fast.
Work for python and typescript project.
More language will be supported in the future.
You can also fork and add more language support yourself.

## Installation

```bash
cargo build --release
```

## Usage

For generate settings.json for python project:

```bash
./target/release/vs-set python
```

For generate settings.json for typescript project:

```bash
./target/release/vs-set typescript
```

## Release

Download the exe from github release.
Put the exe folder in your path.
